# 有型微信小程序工程

这个目录是“有型”的微信小程序工程骨架，用于验证问卷诊断、推荐报告、本地保存和分享入口。

## 导入方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本目录 `miniprogram`。
4. AppID 可先选择测试号，或使用 `touristappid` 体验。
5. 编译后从首页进入“完整诊断”“发型定位”或“身形版型”。

## 当前能力

- 按性别表达、年龄阶段、脸型、肤色、体型、肩线、腰线、腿型、发量、发质、场景、季节和目标效果收集结构化信息。
- 根据年龄阶段自动切换题目模板：宝宝、儿童、校园、职场、中年、银发不会共用同一套问题。
- “发型定位”只输出发型相关建议，不混入完整穿搭判断。
- “身形版型”聚焦肩线、腰线、腿型、版型和场景，不再使用含糊的比例修饰表达。
- 报告保存在微信本地缓存，不上传用户数据。

## 后续开发建议

- 将网页原型中的高阶 UI 逐步迁移到小程序页面。
- 增加“术语科普”页，解释柔和轮廓、清晰轮廓、量感、腰线、肩线等概念。
- 补充不同年龄与性别表达下的推荐知识库，例如宝宝亲肤面料、儿童活动场景、大学生校园穿搭、年轻职场、成熟职场、中年男性、银发舒适穿搭。
