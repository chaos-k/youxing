# 有型产品方案

## 产品定位

有型不是单纯的穿搭灵感集合，而是一个轻量形象顾问工具。它先把用户情况结构化，再基于服装设计、形象设计和日常场景给出可执行建议。

## 目标用户

- 不知道自己适合什么发型、颜色、版型的人。
- 想快速获得通勤、面试、约会、校园、亲子、银发等场景建议的人。
- 希望把“穿得好看”拆成可理解规则的人。
- 后续可扩展到个人衣橱、导购、真人顾问或 AI 报告润色。

## 诊断入口

| 入口 | 目标 | 题目范围 |
| --- | --- | --- |
| 完整诊断 | 输出综合形象和穿搭建议 | 性别、年龄、脸型、体型、肤色、发型、场景、正式度、目标效果 |
| 发型定位 | 只解决发型方向 | 脸型、颈长、发量、发质、年龄气质 |
| 身形版型 | 解决版型和比例选择 | 肩线、腰线、腿型、身高区间、场景、正式度 |

## 年龄阶段

| 阶段 | 典型需求 | 推荐关注点 |
| --- | --- | --- |
| 宝宝 0-3 | 安全、亲肤、便于照护 | 面料、穿脱、保暖、防勒 |
| 儿童 4-12 | 校园、户外、耐穿 | 耐磨、活动量、耐脏、颜色活力 |
| 青少年 13-17 | 校园和成长阶段 | 清爽、不过度成熟、舒适活动 |
| 大学生 18-22 | 上课、社交、实习 | 可尝试、低成本、多场景切换 |
| 初入职场 23-30 | 通勤、面试、约会 | 专业感、个人风格、质感入门 |
| 成熟职场 31-40 | 稳定气场和品质 | 材质、剪裁、成熟度、场合适配 |
| 中年 41-55 | 精神度和体面感 | 版型、材质、颜色克制、比例清晰 |
| 银发 56+ | 舒适与精神 | 轻便、安全、气色、易打理 |

## 结果输出

- 风格路线：例如通勤质感、校园清爽、儿童舒适、银发精神。
- 发型建议：长度、刘海、层次、卷度、避雷点。
- 色彩建议：主色、内搭色、发色方向、禁忌色。
- 版型建议：上装、下装、外套、鞋履、配饰。
- 避雷提醒：不用缺陷化语言，采用中性修饰表达。
- 推荐依据：把专业术语转成用户能理解的解释。

## 产品原则

- 先做可解释的规则判断，再用 AI 做表达润色。
- 不用攻击性审美语言，不评价身材缺陷。
- 不把所有人塞进同一张问卷，按年龄和场景调整题目。
- 不让发型入口混入穿搭问题，避免用户路径失焦。
